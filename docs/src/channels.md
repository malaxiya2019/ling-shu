# 通道配置

## Telegram

```env
LINGSHU_TELEGRAM_BOT_TOKEN=your_bot_token
```

## 飞书

```env
LINGSHU_FEISHU_APP_ID=cli_xxx
LINGSHU_FEISHU_APP_SECRET=xxx
```

## QQ

```env
LINGSHU_QQ_BOT_TOKEN=your_qq_token
```

## 微信

```env
LINGSHU_WECHAT_TOKEN=your_token
LINGSHU_WECHAT_APP_ID=wx_xxx
LINGSHU_WECHAT_APP_SECRET=xxx
LINGSHU_WECHAT_ENCODING_AES_KEY=xxx
```
