# ── Lingshu Helm Chart — Post-Install Notes ──

🚀 **Lingshu Agent System** v{{ .Chart.AppVersion }} is deployed!

────────────────────────────────────────────────

## Verify Deployment

  kubectl get pods -l app.kubernetes.io/name={{ include "lingshu.name" . }}

## Test API

  # Port-forward to localhost
  kubectl port-forward svc/{{ include "lingshu.fullname" . }} 8080:{{ .Values.service.httpPort }}

  # Health check
  curl http://localhost:8080/health

  # API Documentation
  curl http://localhost:8080/docs

  # Swagger UI
  curl http://localhost:8080/docs/swagger

## Federation (Multi-Replica)

  {{- if .Values.federation.enabled }}
  This deployment has federation **enabled** with {{ .Values.federation.topology }} topology.

  # Check federation status
  curl http://localhost:8080/v1/federation/status

  # List online nodes
  curl http://localhost:8080/v1/federation/nodes

  {{- if eq .Values.workloadType "StatefulSet" }}
  Pods discover each other via DNS SRV records from the headless service `{{ include "lingshu.fullname" . }}-federation`.
  {{- else }}
  For production federation, consider setting `workloadType: StatefulSet` for stable pod DNS names.
  {{- end }}
  {{- else }}
  Federation is **disabled**. Set `federation.enabled: true` to enable cross-cluster communication.
  {{- end }}

## Accessing WebUI

  # After port-forwarding, open in browser:
  #   http://localhost:8080/admin     (Server-rendered admin panel)
  #   http://localhost:8080/webui/    (WASM-based panel — requires trunk build)

## Monitoring

  {{- if .Values.monitoring.enabled }}
  Prometheus ServiceMonitor is configured. Metrics are scraped at `{{ .Values.monitoring.serviceMonitor.interval }}`.
  {{- else }}
  Enable monitoring with `monitoring.enabled: true` (requires kube-prometheus-stack).
  {{- end }}

## Configuration

  # View current configuration
  kubectl describe configmap {{ include "lingshu.fullname" . }}-config

  # Override values
  helm upgrade {{ .Release.Name }} . --set replicaCount=5 --set federation.topology=HubSpoke

## Useful Commands

  # Scale
  kubectl scale --replicas=5 deployment/{{ include "lingshu.fullname" . }}

  # Logs
  kubectl logs -l app.kubernetes.io/name={{ include "lingshu.name" . }} --tail=50 -f

  # Uninstall
  helm uninstall {{ .Release.Name }}

  # List installed versions
  helm list
